# Restaurant Order Service

Spring Boot REST API for managing restaurant orders.

## Tech Stack

- Java 21
- Spring Boot
- Maven
- REST APIs
- JSON

---

## API Endpoints

GET /health

GET /orders

GET /orders/{id}

POST /orders

Clone the repository

Run the application

mvn spring-boot:run

Open in browser

http://localhost:8080/orders
